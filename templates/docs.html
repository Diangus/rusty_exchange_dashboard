Exchange Class

Synchronous client for connecting to the exchange trading system.

This class provides a synchronous interface to the exchange's execution and market data endpoints. It handles authentication, order management, and real-time market data access.

Constructor

__init__(self, username: str, password: str) -> None

Initialize the Exchange client with authentication credentials.

Parameters:

username (str): Username for authentication with the exchange
password (str): Password for authentication with the exchange

is_connected()

is_connected(self) -> bool

Check if the exchange connection is active.

Returns:

bool: True if both execution and market data connections are active, False otherwise

connect()

connect(self) -> None

Establish connection to the exchange and authenticate.

This method starts background threads for execution and market data connections, waits for connections to be established, and performs authentication.

Raises:

AssertionError: If already connected
Exception: If connection fails or authentication fails

get_outstanding_orders()

get_outstanding_orders(self, instrument: str) -> dict[str, Orders]

Shows all outstanding active orders for the connected client. Active orders are orders that have not been completely filled or cancelled.

Parameters:

instrument (str): The instrument identifier

Returns:

dict[str, Orders]: Dictionary with 'asks' and 'bids' keys containing all outstanding orders for the specified instrument from all clients that have not been completely filled or cancelled. Each order object contains:
  • client_name (str): Username/client identifier who placed the order
  • instrument (str): Trading instrument identifier
  • leaves_qty (int): Remaining quantity available on the order
  • price (float): Order price level
  • request_id (str): Unique identifier for the original order request
  • side (str): Order side indicator ("Buy" or "Sell")
  • timestamp (str): ISO 8601 timestamp of when the order was last updated

get_orderbook()

get_orderbook(self, instrument: str) -> dict[str, Orders]

Get the current order book for a specific instrument.

Parameters:

instrument (str): The instrument identifier

Returns:

dict[str, Orders]: Dictionary with 'asks' and 'bids' keys containing lists of order objects. Orders are sorted by price (asks ascending, bids descending) then by timestamp. Each order object contains:
  • client_name (str): Username/client identifier who placed the order
  • instrument (str): Trading instrument identifier
  • leaves_qty (int): Remaining quantity available on the order
  • price (float): Order price level
  • request_id (str): Unique identifier for the original order request
  • side (str): Order side indicator ("Buy" or "Sell")
  • timestamp (str): ISO 8601 timestamp of when the order was last updated

get_trades()

get_trades(self) -> list[dict[str, Any]]

Get all trades that have occurred since the last call to this method.

Returns:

list[dict[str, Any]]: List of trade dictionaries. Each trade contains:
  • buyer (str): Username of the buying party
  • buyer_order_id (str): Request ID of the buyer's original order
  • instrument (str): Trading instrument identifier
  • price (float): Execution price of the trade
  • seller (str): Username of the selling party
  • seller_order_id (str): Request ID of the seller's original order
  • timestamp (str): ISO 8601 timestamp of when the trade was executed
  • volume (int): Quantity traded in this execution

get_positions()

Example Usage

# Create client instance
exchange = Exchange("username", "password")

# Connect to exchange
exchange.connect()

# Check connection
if exchange.is_connected():
    print("Connected to exchange")

# Place an order
order_id = exchange.insert_order("AAPL", "Buy", 100, 150.50, "GFD")
print(f"Order placed: {order_id}")

# Get orderbook
orderbook = exchange.get_orderbook("AAPL")
print(f"Asks: {orderbook['asks']}")
print(f"Bids: {orderbook['bids']}")

# Get positions
positions = exchange.get_positions()
print(f"Positions: {positions}")

# Cancel the order
exchange.cancel_order(order_id)

# Cancel all orders
exchange.cancel_all_orders()
